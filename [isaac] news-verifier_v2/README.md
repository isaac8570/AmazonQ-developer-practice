# 🔍 FactCheck AI - 뉴스 검증 플랫폼

실시간 웹 검색과 AI를 활용한 가짜뉴스 검증 서비스

## 🚀 주요 기능

### 📊 **지능형 신뢰도 분석**
- **다차원 신뢰도 평가**: 출처 신뢰도, 검색어 연관성, 날짜 신선도, 제목 품질을 종합 분석
- **15개 뉴스 분석**: 기존 7개에서 15개로 확대하여 더 정확한 신뢰도 판단
- **관련도순 정렬**: 검색어와 가장 관련성 높은 뉴스부터 우선 표시
- **스마트 필터링**: 신뢰도별 필터링으로 원하는 품질의 뉴스만 선별

### 🗺️ **인터랙티브 마인드맵**
- **방사형 구조**: 중심 키워드 → 관련 키워드 → 뉴스 노드 순으로 직관적 배치
- **드래그 & 드롭**: 모든 노드를 자유롭게 이동 가능, 연결된 노드들이 함께 움직임
- **확대/축소**: 마우스 휠 및 컨트롤 버튼으로 0.3배~3배 확대/축소
- **키워드 연결**: 뉴스 제목에서 자동 추출한 키워드로 관련성 시각화

### 📈 **시간대별 뉴스 트렌드**
- **시간순 정렬**: 오래된 기사부터 최신 기사까지 시간 흐름 파악
- **통계 정보**: 총 뉴스 개수, 날짜 확인된 뉴스, 발행일 범위 표시
- **클릭 가능**: 표의 각 행을 클릭하여 원본 기사 바로 이동

### 🎯 **실시간 검증**
- **자연어 질문**: 일상 언어로 뉴스 진위 확인
- **다중 출처 분석**: 여러 신뢰할 수 있는 출처에서 정보 수집
- **AI 분석 결과**: 종합 분석, 주요 발견사항, 권장사항 제공

## 🧮 신뢰도 점수 시스템

### **점수 구성 (총 100점)**

#### 1. **출처 신뢰도 (35%)**
- **High 신뢰도**: 35점 (KBS, MBC, SBS, 연합뉴스 등)
- **Medium 신뢰도**: 20점 (네이버, 다음, 매일경제 등)
- **Low 신뢰도**: 5점 (기타 출처)

#### 2. **검색어 연관성 (30%)**
- 검색어와 뉴스 제목의 단어 매칭률 분석
- 정확한 검색어 포함 시 보너스 점수
- **연관성이 낮으면 점수 대폭 하락**

#### 3. **날짜 신선도 (15%)**
- 1일 이내: 15점 | 1주 이내: 12점
- 1달 이내: 9점 | 3달 이내: 6점 | 그 이상: 3점

#### 4. **제목 품질 (10%)**
- 적절한 길이 (10-100자)
- 특수문자 남용 없음
- 광고성 키워드 배제

#### 5. **기타 요소 (10%)**
- 상세 설명 존재 여부 (+10점)
- 출처 다양성 보너스 (최대 8점)
- 뉴스 개수 조정 (3개 미만 -10점, 10개 이상 +5점)

### **신뢰도 등급**
- **80점 이상**: 높은 신뢰도 ✅
- **60-79점**: 보통 신뢰도 ⚠️
- **40-59점**: 낮은 신뢰도 ❌
- **40점 미만**: 매우 낮은 신뢰도 🚫

### **특별 검증 로직**
- 고품질 뉴스(High 신뢰도 + 높은 연관성)가 없으면 **최대 45점으로 제한**
- 검색어와 무관한 뉴스들의 점수 인플레이션 방지

## 🛠 기술 스택

- **Backend**: Node.js, Express.js
- **Frontend**: Vanilla JavaScript, HTML5, CSS3, D3.js
- **시각화**: D3.js Force Simulation, Interactive Mind Map
- **MCP Integration**: Exa Search MCP
- **Deployment**: AWS (EC2, S3, CloudFront)

## 📦 설치 및 실행

1. **의존성 설치**
```bash
npm install
```

2. **환경 변수 설정**
```bash
cp .env.example .env
# .env 파일에서 API 키 설정
```

3. **서버 실행**
```bash
npm start
# 또는 개발 모드
npm run dev
```

4. **브라우저에서 접속**
```
http://localhost:3003
```

## 🎮 사용법

### **마인드맵 조작**
- **확대/축소**: 마우스 휠 또는 우측 상단 버튼 (+, -, ⌂)
- **노드 이동**: 마우스로 드래그
- **연결된 이동**: 중심 노드 드래그 시 키워드들이 함께, 키워드 드래그 시 관련 뉴스들이 함께 움직임
- **기사 보기**: 뉴스 노드 클릭

### **뉴스 목록**
- **필터링**: 전체/높은 신뢰도/보통 신뢰도/낮은 신뢰도
- **관련도 확인**: 각 뉴스의 관련도 점수 표시
- **상세 정보**: 신뢰도, 발행일, 도메인, 설명 포함

### **타임라인**
- **시간순 정렬**: 오래된 기사부터 최신 순
- **통계 정보**: 날짜별 뉴스 분포 확인
- **원클릭 이동**: 행 클릭으로 원본 기사 이동
